##  Git
1. 깃(Git) 이란? 
  - `Git`은 버전 관리를 위한 도구 -`VCS`  
  - 코드를 작성한 모든 `History`가 저장되기 때문에 오류가 발생할 경우 롤백에 유용
  - `Brench`와`merge`가 가능해 여러 사람간 협업에도 자주 활용  
  (여러 사람이 각자 복사본을 작업해서 다시 한 곳에 합치는 기능)
  
2. GitHub
  - 깃을 설치 없이 웹에서 사용 가능합니다.  
  - `GitHub`을 통해 `Markdown`을 배워봅시다.
